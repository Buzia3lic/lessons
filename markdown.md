# Заголовок первого уровня обозначается # #  
## Заголовок второго уровня обозначается ## ##
### и так далее... ###

## Цитаты обозанчаются знаком >
> первый уровень
>> второй
>>> и так далее...
## Списки
Списки без номеров обозначаются *, + и -.
* один *
+ два +
- три -

С номерами просто номер и точка

1. один
2. два
3. три
